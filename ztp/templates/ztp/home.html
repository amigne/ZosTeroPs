{% blocktrans %}ZosTeroPs manages the ZTP (Zero Touch Provisioning) of your network devices.{% endblocktrans %}

{% blocktrans %}The system contains different parts that permit to customize the process depending on the vendor and the platform of your network devices.{% endblocktrans %}

  • {% if perms.ztp.list_ztpscript %} {% trans 'ZTP Scripts' %} {% else %} {% trans 'ZTP Scripts' %} {% endif %} {% blocktrans %}collects your ZTP bootstrap scripts that are first downloaded by the network devices. Multiple scripts can be defined to support multiple platforms with different bootstrap conditions. All scripts are text-based (not binary) that can define variable substitution. {% endblocktrans %}
  • {% if perms.ztp.list_config %} {% trans 'Configurations' %} {% else %} {% trans 'Configurations' %} {% endif %} {% blocktrans %} defines your configuration templates that can be copied by the ZTP script to your device, if it supports this possibility. Splitting the bootstrap and the configuration permits to have a more generic bootstrap script and custom configurations for specific platforms or for specific devices. {% endblocktrans %}
  • {% if perms.ztp.list_firmware %} {% trans 'Firmwares' %} {% else %} {% trans 'Firmwares' %} {% endif %} {% blocktrans %} collects all the firmwares (softwares) that can be downloaded by the managed devices. {% endblocktrans %}
  • {% if perms.ztp.list_platform %} {% trans 'Platforms' %} {% else %} {% trans 'Platforms' %} {% endif %} {% blocktrans %} manages the categories of the network devices of a given vendor. {% endblocktrans %}
  • {% if perms.ztp.list_vendor %} {% trans 'Vendors' %} {% else %} {% trans 'Vendors' %} {% endif %} {% blocktrans %} manages the vendors of the network devices. {% endblocktrans %}
  • {% trans 'Logs' %} {% trans '(not implemented)' %} {% blocktrans %} displays the logs of ZosTeroPs. {% endblocktrans %}
